Retention

Retention

Choose how many recent daily, monthly, and yearly snapshots to keep automatically.

Example: 3 recent days keeps today, yesterday, and the day before. Monthly retention keeps one snapshot per month. Yearly retention keeps one snapshot per year.

Automation

Automation

Set the backup schedule, stale-backup reminders, and cloud health checks.

This creates Windows Scheduled Tasks so the PC can run backups automatically and remind the user when backups have gone stale. Manual backups still work even if you do not install the tasks.
